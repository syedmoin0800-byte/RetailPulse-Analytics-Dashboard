# RetailPulse Analytics Dashboard 📊

## Project Overview
RetailPulse is an interactive retail sales analytics dashboard developed using Python and Streamlit. The project converts raw retail sales data into meaningful insights through visualizations and interactive filters.

## Features
- Sales KPI Analysis (Total Sales and Total Orders)
- Sales Analysis by Category
- Top 10 Selling Products
- Region-wise Sales Analysis
- Customer Segmentation Analysis
- Sub-Category Performance Analysis
- Monthly Sales Trend Analysis
- Yearly Sales Trend Analysis
- Top Customers Analysis
- Interactive Region and Category Filters
- Dataset Information and Missing Value Analysis

## Technologies Used
- Python
- Pandas
- Streamlit
- VS Code
- Jupyter Notebook
- GitHub

## Dataset Description
The dataset contains 9,986 records and 18 columns, including:
- Order Details
- Customer Details
- Product Information
- Category and Sub-Category
- Region Information
- Sales Data

## Dashboard Modules
1. Dataset Overview
2. Sales Analytics
3. Customer Segmentation
4. Product Performance Analysis
5. Sales Trend Analysis
6. Interactive Filtering

## Dataset Limitation
The dataset does not contain a Profit column. Therefore, profit analysis was not included in the project.
